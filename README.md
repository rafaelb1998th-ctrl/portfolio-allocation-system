# Portfolio & allocation system (Hedge)

Work in progress (2025–present). Personal **data-driven allocation** project: multiple inputs (datasets / sleeves / constraints), **rule-based** decisions, and structured portfolio outputs.

## Highlights

- Modular layout for **datasets and asset classes** (separate concerns: data, signals, allocation, reporting).
- **Rule-based** allocation and portfolio structuring (explicit policies instead of ad-hoc spreadsheets).
- Designed for **repeatable** runs, audit-style logs, and growth without a single giant script.

## What you will not find here

The full codebase, broker integrations, API credentials, and live `state/` data stay **private** (security and personal data). This public repo is a **description + boundaries** so recruiters can see the problem you are solving without exposing secrets.
