# Public snapshot

This repository is a **sanitized** copy of a personal portfolio automation project.

**Not included (by design):** `.env`, `state/`, `archive/`, `logs/`, `venv/`, runtime data, or backups.

**Before running anything locally:** copy `.env.template` to `.env` and fill in your own API keys. Never commit `.env`.

Symlinks under `backups/` or `out/` may be broken here; they pointed at `archive/` which is omitted from the public tree.
